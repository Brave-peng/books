   - 以地方县志之类相关材料为依据，讲述了古代基层政治斗争的故事，上至郡县交互，下至乡绅管理，对中国特色的政治平衡之道有非常有趣的描写，让人惊叹古人的智慧。
   - 视角聚焦于普通史书看不到的地方，读后阅读视野更加开阔。
   - 马亲王很会写故事，读起来很有趣。
